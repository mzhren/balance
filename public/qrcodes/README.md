# 赞赏二维码说明

## 添加收款二维码

请将您的收款二维码图片放置在此目录下：

1. **微信收款码**：`wechat-pay.png`
   - 建议尺寸：400x400 像素或更大
   - 格式：PNG 或 JPG

2. **支付宝收款码**：`alipay.png`
   - 建议尺寸：400x400 像素或更大
   - 格式：PNG 或 JPG

## 获取收款码方法

### 微信收款码
1. 打开微信 APP
2. 点击右上角 "+" → "收付款" → "二维码收款"
3. 点击"保存收款码"
4. 将图片重命名为 `wechat-pay.png` 并放入此目录

### 支付宝收款码
1. 打开支付宝 APP
2. 点击"收钱"
3. 点击"保存图片"
4. 将图片重命名为 `alipay.png` 并放入此目录

## 注意事项

- 确保图片清晰可扫描
- 建议使用正方形图片以保持最佳显示效果
- 图片文件名必须完全匹配（区分大小写）
- 如果暂时没有收款码，页面会显示占位符提示
